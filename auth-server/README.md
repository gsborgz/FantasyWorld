# Sereno API - Visão Geral

O Sereno API é uma aplicação backend desenvolvida em NestJS para gerenciamento de psicólogos e pacientes. O sistema oferece endpoints REST, internacionalização (i18n), segurança e validação de dados, integrando-se ao frontend sereno-app.

Principais funcionalidades:

- Cadastro e autenticação de usuários (pacientes e psicólogos)
- Listagem e busca de psicólogos
- Gerenciamento de sessões de autenticação
- Internacionalização (português e inglês)
- Validação e tratamento de erros padronizados

Para mais detalhes, consulte o diretório `docs/`
